# Open-Source Readiness

This folder contains public-facing maturity evidence for Agent Forge. It is
separate from `docs/study-pack/` so the study path stays focused on reading the
runtime code, while this folder answers project-readiness questions.

Read in this order:

1. `benchmark-summary.md`
2. `ablation-notes.md`
3. `docker-sandbox-extension-plan.md`
4. `provider-comparison.md`

The documents are intentionally lightweight and reproducible. They point to
code, eval cases, committed run artifacts, and verification commands rather
than claiming production traffic or hosted-service metrics.
